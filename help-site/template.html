Archive Spreadsheet Guide

How to describe items in the archive using the spreadsheets.

Who this is for

This guide is for anyone recording items in the archive — regular archive users as well as donating institutions contributing their own records. It explains what each column in the spreadsheets means and how to fill it in.

How the spreadsheets work

The archive is described using a set of spreadsheets. Each spreadsheet covers one kind of thing — for example, the Resources sheet lists the items themselves (photos, recordings, objects), while the People, Places, and Organisations sheets list the people, places, and institutions those items refer to.

Each row is one item. Each column is a piece of information about that item. Some columns hold plain text you type in; others link to a row in another sheet (for example, a resource's creator links to a row in the People sheet). Where a column links to another sheet, it is noted below.
